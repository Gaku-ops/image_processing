1. HSV手法（指定した色をカッチリ抜き出す）
HSVは、画像を「色合い（Hue）」「鮮やかさ（Saturation）」「明るさ（Value）」の3つの要素に分けて考える手法です。

仕組み: 「この色からこの色まで」という絶対的な範囲（しきい値）を人間が固定で指定します。
メリット: 非常に直感的です。「鮮やかな緑だけ」や、逆に「熟した赤いトマトだけ」など、狙った色を正確に抜き出すことができます。
デメリット: 光の変化に非常に弱いです。例えば、晴れの日にピッタリ合うようにスライダーを調整しても、雲がかかって日陰になった瞬間に「明るさ（V）」が指定範囲から外れてしまい、植物を全く認識できなくなる（真っ黒になる）ことがあります。
2. ExG手法（環境光に自動で適応する）
ExG（Excess Green：拡張緑色指標）は、農業分野特有の数式手法です。ピクセルの色を「緑×2 － 赤 － 青」という計算式にかけて、「とにかく緑色が強い部分」を白く浮かび上がらせます。

仕組み: 浮かび上がらせた後に「大津の二値化」というアルゴリズムを組み合わせるのが一般的です（今回のシステムもそうなっています）。大津の二値化は、その時の画像の明るさに合わせて、自動で最適な白黒の境界線を引いてくれます。
メリット: 日向や日陰などの環境光の変化にめちゃくちゃ強いです。太陽の光が変わっても、自動でしきい値を計算し直すため、安定して植物（緑色）と土（茶色）を分離し続けてくれます。GUIのスライダー（Offset）は、その自動計算された基準線に対する微調整としてのみ機能します。
デメリット: 「緑色」を抽出することに特化しているため、赤い実や黄色い花などを抽出することには使えません。また、土の表面に緑色のコケが生えていると、それも植物として抽出してしまいます。
💡 現場での使い分けの目安
ExG： ドローンや屋外の自動走行ロボットなど、太陽光がコロコロ変わる屋外の畑で、雑草や作物の葉の面積（生育状況）を調べる場合。
HSV： 工場内や密閉型の植物工場など、常に一定のLED照明が当たっている環境で、特定の色だけを厳密に検査したい場合。
